# Sistema GMB Crush — Pack operativo completo v4

Este repositorio contiene la versión operativa compacta del sistema GMB Crush para crear, refactorizar, publicar y validar webs locales orientadas a GBP, Local Pack, SEO orgánico y AI Overviews.

## Orden de lectura

1. `00_BASE_DEL_SISTEMA_GMB.md` — doctrina transversal.
2. Elegir uno de los cuatro escenarios:
   - `01_ESCENARIO_SIN_WEB_Y_SIN_GBP.md`
   - `02_ESCENARIO_SIN_WEB_Y_CON_GBP.md`
   - `03_ESCENARIO_CON_WEB_Y_SIN_GBP.md`
   - `04_ESCENARIO_CON_WEB_Y_CON_GBP.md`
3. `05_IMPLEMENTACION_TECNICA.md` — estándar técnico de construcción.
4. `06_BIBLIOTECA_DE_PROMPTS_GMB_CRUSH.md` — prompts copy-ready para ejecutar cada fase.
5. `REVISION_DE_COBERTURA_GMB.md` — informe de control doctrinal.

## Regla de uso

La base define los principios.  
El escenario define la hoja de ruta.  
La implementación técnica define cómo producir la web.  
La biblioteca de prompts permite ejecutar cada fase sin improvisar.

El output final no es un documento: es una web publicada, conectada o alineada con GBP, con arquitectura local, interlinking, schema, contenido y QA final aprobados.
